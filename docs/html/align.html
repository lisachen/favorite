对齐方式

垂直居中 水平居中 垂直水平居中 两端对齐

垂直居中：

方法一：使用padding

HTML:

<div class="parent">
	<div class="details">内容内容</div>
</div>

CSS:

.details {
	padding-top:30px;
	padding-bottom:30px;
}

优点：兼容所有浏览器，支持所有元素，简单易懂

缺点：如果父级容器固定，达不到居中效果

方法二：单行文字垂直居中(使用行高)

HTML:

<div class="vertical">内容</div>

CSS:

.vertical{
	height:30px;
	line-height:30px;
}

优点：兼容所有浏览器；

缺点：只适合单行文字和图片

方法三：线盒型

 
内容

<div class="parent">

优点：兼容所有浏览器

缺点：需要固定的宽高，内容无法自适应；需要加一个空标签

水平居中：

方法一：margin: auto+width

HTML:

<div class="details">content</div>

CSS:

.details {
	width: 200px;
	margin: 0 auto;
}

优点：支持所有浏览器运行

缺点：需要固定宽度

方法二：text-align:center

HTML:

<div class="details"><span>content</span></div>

CSS:

.details {
		text-alidn:center;
}

优点：支持所有浏览器运行,简单易懂

缺点：只适应内联标签，文字需重新设置对齐方式


垂直水平居中

方法一：使用绝对定位

HTML:

<div class="vertical">内容</div>

CSS:

.vertical {
	width: 元素高度;	
	height: 元素高度;
	position: absolute;
	top: 50%;
	left：50%;
	margin-top: -元素高度/2;
	margin-left: -元素宽度/2;
}

优点：兼容所有浏览器；结构简单明了，不需增加额外的标签

缺点：需要固定的宽高；绝对定位脱离了文档流

方法四：-webkit-transform: translate(-50%,-50%);

translate 实现垂直居中

HTML:

<div class="parent">
	<div class="translate-center">
		translate 实现垂直居中
	</div>
</div>

CSS:

.box-translate .parent{
	position:relative;
	height: 100px;
	border:1px dashed #ddd;
	background-color:#fefefe;
}
.translate-center{
	position: absolute;
	left: 50%;
	top: 50%;
	background-color:#eee;
	-webkit-transform: translate(-50%, -50%) ;
	-moz-transform: translate(-50%, -50%) ;
	-ms-transform: translate(-50%, -50%) ;
	-o-transform: translate(-50%, -50%) ;
	transform: translate(-50%, -50%) ;
}

优点：内容高度可变，代码量小

缺点：不支持IE8，需要写厂商前缀，会和其他transform样式有冲突
