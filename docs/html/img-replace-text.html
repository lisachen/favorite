用图片替换文本的方法

公共样式

.text{
	display:block;
	width:100px;
	height:100px;
}

1.使用text-index负值

文字

HTML:

<a class="method-01 text">文字</a>

CSS:

.method-01{
	background:url(../images/test/text_bg.jpg) no-repeat;
	text-indent:-9999em;
	overflow:hidden;
}

这种方法简单易懂而且支持阅读器之类浏览网页。

2.使用line-height

文字

HTML:

<a class="method-02 text">文字</a>

CSS:

.method-02{
	background:url(../images/test/text_bg.jpg) no-repeat;
	line-height:100;
	overflow:hidden;
}

这种方法简单易懂而且支持阅读器之类浏览网页。

3.设置文本不显示

文字

HTML:

<a class="method-03 text"><span>文字</span></a>

CSS:

.method-03{
	background:url(../images/test/text_bg.jpg) no-repeat;
}
.method-03 span{
	display:none;
}

就是当图片无法显示时，将导致这个区域没有任何内容;使用 display:none 的方式隐藏的内容，不利于阅读器浏览网页，因此，应该尽量避免使用。

4.设置高度为0

文字

HTML:

<a class="method-04 text">文字</a>

CSS:

.method-04{
	padding-top:100px;
	height: 0px;
	font-size: 0;
	overflow: hidden;
	background:url(../images/test/text_bg.jpg) no-repeat;	
}

需要设置一个与替换图片一样的大小的padding值。

5.用图片盖住文字

文字

HTML:

<a class="method-05 text"><span></span>文字</a>

CSS:

.method-05 {
	position: relative; 
}
.method-05 span {
	position: absolute; 
	width: 100%; 
	height: 100%; 
	background:url(../images/test/text_bg.jpg) no-repeat;
}

这种方法的原理是用一个空标签放背景图片并覆盖在文字内容上面，图片必须是不透明的。优点：解决浏览器禁用图像时的空白问题
