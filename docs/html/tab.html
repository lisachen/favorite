标签（Tab）切换的实现方法

普通切换-内容全部加载（安排顺序对应）

  • 标签1
  • 标签2
  • 111...
  • 222...

HTML:

<div class="tab module">
	<ul class="tab-nav hd">
		<li class="cur">标签1</li>
		<li>标签2</li>
	</ul>
	<ul class="tab-content bd">
		<li class="tab-pane cur">
			111...
		</li>
		<li class="tab-pane">
			222...
		</li>
	</ul>
</div>

CSS:

.tab-nav {
	position: relative;
	overflow: hidden;
}
.tab-nav li {
	float: left;
	border:1px solid #eee;
	border-bottom:none;
	margin-right:8px;
	padding:0 20px;
	line-height:30px;
	font-size:14px;
	cursor:pointer;
}
.tab-nav .cur{
	background-color:#FAFAFA;
}
.tab-content{
	margin-top: -1px;
	border:1px solid #eee;
	background-color:#FAFAFA;
	height:100px;
}
.tab-content .tab-pane{
	display:none;
}
.tab-content .cur {
	display:block;
}

Jquery:

$('.tab-nav li').click(function() {
	var _this=$(this),
		s=_this.index(),
		$pane=_this.parents('.tab').first().find('.tab-pane').eq(s);
	a_s_r(_this,'cur');
	a_s_r($pane,'cur');
});
function a_s_r(o,c){
	o.addClass(c).siblings().removeClass(c);	
}

注：

如果想鼠标经过切换，将jquery中的click事件换成hover事件。
如果切换的内容是用js动态处理的，就不能使用display:none，可以换成visibility:hidden。

普通切换-内容全部加载（更具ID对应）

HTML:

<div class="tab module">
	<ul class="tab-nav hd">
		<li class="cur"><a href="#tab01">标签一</a></li>
		<li><a href="#tab02">标签二</a></li>
	</ul>
	<ul class="tab-content bd">
		<li class="tab-pane cur" id="tab01">
			111...
		</li>
		<li class="tab-pane" id="tab02">
			222...
		</li>
	</ul>
</div>

Jquery:

$(".tab-nav li").click(function() {
	var	_this=$(this),
		$pane=$(_this.find('a').attr('href'));
	a_s_r(_this,'cur');
	a_s_r($pane,'cur');
});
function a_s_r(o,c){
	o.addClass(c).siblings().removeClass(c);	
}
